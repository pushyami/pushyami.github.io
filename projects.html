Projects

Here are some projects I have during my undergraduation. Few of them are sub-projects part of a whole, some of them are part of course-work and others self-interest projects.

Global and Direct illumination separation (sub-project)

  • Implemented and improvised the fast separation algorithm in order to remove specular reflections on images in which features are not present, as part of Anterior Segment Imaging Project.

NLP Parser

  • Given training data, created the tokenizer, modeled the language(using n-grams), extracted the POS tags and chunking rules. Applied the above extracted rules to testing data, if unseen words are found, used different smoothing techniques, viterbi and morph analysis(for Hindi and Telugu languages) to POS tag and chunk those unseen words. Finally implemented the CYK parser to predict the phrase structure of a sentence.

Database Management System

  • Built a database management system which includes populating the database, allotting pages, creating cache, parsing the query and executing the command. Built my own SQL parser.

Cassandra Browser Tool

  • Created a Cassandra Browser tool for browsing namespaces and data in Cassandra using python. Created a database server with keyspaces and column families and a user interface to pass queries in order to retrieve information about the keyspaces. Implemented all functionalities with respect to key spaces and column families.

Virtualization Orchestration Layer

  • Implemented a fabric that coordinates the provisioning of compute and storage resources by negotiating with a set of hypervisors running across physical servers in the data center.

TA portal

  • Built the Teaching Assistant Portal for IIIT Hyderabad with improved interfaces and additional functionalities. It is an interface through which students can apply for the post of a Teaching Assistant and the instructors can choose assistants for their courses based on a variety of characteristics like GPA of the student or previous experience of the student etc.

OpenGL Pipeline

  • Implemented a coordinate tranformation pipeline, similar to the one in OpenGL, for calculating projected 2D coordinates of different objects.

3D Adventure Game

  • Created a 3D adventure multi-player game using OpenGL library in C++ that simulates a 3D world. Rendered all the physical elements such as water, sky, a 3D human body and an auto generated path for the player to move. Implemented different camera angles and views. Applied all laws of physics such as collisions, friction and gravity as well.

Unix Shell

  • Implemented a bash like shell with I/O redirection, background process management, signal handling, piping of different commands using the concepts of fork, exec and signals.
