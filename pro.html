Research Projects

Master's Thesis: Ontology of Tennis

This thesis aims at formalizing and modeling the complete concept of tennis by designing graph grammars which depict the flow of the game adhering to the rules set by ITF. We have collected and created a huge vocabulary of tennis terms which are defined by various paths and trees in the grammar. The event trees are then used to create a context free grammar. For the computational ease of use of this ontology, hypergraphs are implemented which encompass the complete ontology created. This ontology is used to implement various applications which can be used for the purpose of coaching. An android app was created to annotate tennis videos, which are then used to extract game statistics and player profiles in real time. Designed and implemented improvised algorithms for player and ball detection with varied and natural backgrounds. Determination of the ball spin and next probable shot detection using ball physics and machine learning techniques. Documentation standards are set, which can be used in practice matches to capture the complete game along with a coach's perspective.

Visiting Researcher: Anterior Segment Imaging

Worked as Computer Vision Engineer on the project Anterior Segment Imaging, which was later published at ARVO and also manifested into a patent.Worked on 3D reconstructing the cornea of the eye (non opaque objects in general) by developing improvised algorithms extending them to transparent, featureless objects. Worked on specular reflection removal in images by separating different components of illumination without loss of information. Worked on data collection, first on the optical bench with cadaver bovine eye. Later on the 3D printed prototype. Trained in laser safety and calculated the safety thresholds for the lasers used in the prototype. Assisted in designing the prototype and 3D printing of parts.
